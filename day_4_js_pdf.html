Part 1: The Functional Array Trio (Immutability)

These methods return new arrays and do not modify the original data.

1. map() - The Transformer
"Transform every element and return a new array"
Purpose:
Transform every element in an array and return a new array of the same length.
INPUT
Array of length N
OUTPUT
New Array of length N
const prices = [10, 20, 30]; const taxedPrices = prices.map((price) => { return price * 1.2; // Add 20% tax });
Result: [12, 24, 36]
2. filter() - The Gatekeeper
"Select elements that pass a condition"
Purpose:
Select a subset of elements that pass a specific Boolean condition (true).
INPUT
Array of length N
OUTPUT
New Array of length 0 to N
const scores = [45, 80, 92, 30, 60]; const passingScores = scores.filter((score) => { return score >= 60; // Only passing grades });
Result: [80, 92, 60]
3. reduce() - The Accumulator
"Aggregate an array down to a single value"
Purpose:
Aggregate an array down to a single value (can be a number, string, object, or array).
INPUT
Array of length N
OUTPUT
Single value (any type)
const cart = [20, 50, 10]; const total = cart.reduce((totalSoFar, currentPrice) => { return totalSoFar + currentPrice; }, 0); // initialValue is 0
Result: 80

Part 2: Object Bridge Methods

Convert parts of an object into an iterable array.

Method Returns Example (user = {name: "A", age: 28})
Object.keys() Array of property names (Strings) ["name", "age"]
Object.values() Array of property values ["A", 28]
Object.entries() Array of [key, value] arrays [["name", "A"], ["age", 28]]

ES5 vs. ES6 (ECMAScript 2015)

Feature ES5 (Old Way) ES6 (Modern Way) Advantage
Variable Scoping var (Function-scoped, hoisted) let and const (Block-scoped) Prevents hoisting bugs and limits variable mutation
Functions function keyword, manual this binding Arrow Functions (=>) Concise syntax; automatically binds this
String Concatenation 'Hello ' + name Template Literals (`Hello ${name}`) Easy variable interpolation and multi-line strings

Key ES6 Features

1. let and const

Block-scoped variables

const PI = 3.14; // Cannot reassign let count = 0; // Can reassign
2. Arrow Functions

Shorthand syntax with lexical this

const add = (a, b) => a + b;
3. Classes

OO-style structure over prototypes

class Dog { constructor(name) { this.name = name; } bark() { console.log(`${this.name} says Woof!`); } }
4. Destructuring

Extract values quickly

const person = { first: 'J', last: 'Doe' }; const { first, last } = person; const [x, y] = [10, 20];
5. Spread & Rest (...)

Expand or bundle elements

// Spread const arr2 = [...arr1, 3, 4]; // Rest function sumAll(...numbers) { return numbers.reduce((t, n) => t + n, 0); }
6. async/await

Cleaner asynchronous code

async function getData() { try { const data = await fetch('/api/data') .then(res => res.json()); return data; } catch (error) { console.error(error); } }
